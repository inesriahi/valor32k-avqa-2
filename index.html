Valor32k-GPT4 Dataset

Dataset Description

Valor32k-GPT4 is a comprehensive audio-visual question answering dataset containing 32,000 carefully curated examples. Each example consists of a video clip, detailed captions, and multiple-choice questions with rephrased answers. The dataset covers various categories and includes questions that can be answered through visual, audio, or audio-visual modalities.
